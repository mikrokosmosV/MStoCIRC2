# External bioinformatics tools are documented here for completeness, but they
# are not installable through pip and therefore are not listed as Python
# requirements:
# - bedtools: required for genome-based ORF sequence extraction
# - diamond: required for canonical-protein background filtering in eval;
#   MStoCIRC2 uses `diamond makedb` and `diamond blastp`
# - FragPipe + DIA-NN: required for search / nonquant / quant workflows
#
# Optional predictor runtimes that are also outside pip resolution:
# - DeepCIP: upstream-managed Linux toolchain with Python 3.8, Perl, Anaconda,
#   ViennaRNA/RNAplfold, and Seqkit
# - DeepCircM6A: bundled in this repository, but still requires the Python
#   package `torch` above
#
# DEA also depends on an external R runtime:
# - R >= 4.3 with Rscript on PATH
# - BiocManager >= 1.30
# - Bioconductor release-compatible versions of limma, DEqMS, proDA, pcaMethods, ROTS
# - CRAN version of rrcovNA >= 0.5-0
#
# Note:
# - FragPipe DIA workflows also require `fragpipe-speclib`, `easypqp`, and
#   `lxml`, but those packages belong in the user-selected FragPipe DIA Python
#   environment rather than the default MStoCIRC2 installation environment.
